# MSC Tracker App

A mobile web application using Flutter and Google Firebase to streamline operations in Motorization Support Company (MSC), 2nd Battalion Singapore Infantry Regiment. The app creates a standardized system for data management and visualization of the vehicle fleet and equipment, providing a simple way for data sharing across the company, which improves operational efficiency and communication.
